### Props

| 参数      | 说明               | 类型                      | 默认值 | 必填 |
| --------- | ------------------ | ------------------------- | ------ | ---- |
| className | 类名               | string                    | 无     | 否   |
| size      | loading 元素的大小 | number                    | 20     | 否   |
| color     | loading 元素的颜色 | string(同 css 中的颜色值) | "#333" | 否   |
